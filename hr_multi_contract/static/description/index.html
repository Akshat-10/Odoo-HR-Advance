# HR Multi Contract Management ## Quick Start Guide ### Installation 1. Copy the module to your addons directory 2. Update the app list in Odoo 3. Install "HR Multi Contract Management" ### Requirements - hr (base HR module) - hr_contract - hr_employee_entended ### Key Features ✓ Create multiple contracts simultaneously ✓ Auto-populate custom fields (Employee Code, Father Name) ✓ Use employee joining dates automatically ✓ Wizard-based interface for quick access ✓ Complete audit trail with multi-contract records ✓ Action button in employee list view ### Usage **Method 1**: Select employees → Action → Create Mass Contracts **Method 2**: Employees → Contracts → Multi Contracts → Create For detailed documentation, see README.md
